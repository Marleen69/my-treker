ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ТРЕКЕРА ПРИВЫЧЕК
=============================================

ВАРИАНТ 1: Использование Google Apps Script (РЕКОМЕНДУЕТСЯ)
------------------------------------------------------------

1. Откройте Google Sheets (sheets.google.com)
2. Создайте новый пустой лист
3. Откройте меню: Extensions > Apps Script
4. Удалите весь код по умолчанию
5. Скопируйте содержимое файла setup_habit_tracker.gs и вставьте в редактор
6. Нажмите кнопку "Save" (Сохранить) или Ctrl+S
7. В меню выберите функцию "setupHabitTracker" и нажмите Run (Выполнить)
8. При первом запуске разрешите доступ к Google Sheets
9. Вернитесь в лист - он будет автоматически настроен!

Что будет создано:
- Заголовок месяца (NOVEMBER)
- Статистика: "Number of Habits" и "Completed Habits Today"
- Таблица с примерами привычек и чекбоксами на 31 день
- Блок статистики с процентами выполнения по дням
- Зеленый график выполнения привычек

ВАРИАНТ 2: Ручная настройка через CSV
--------------------------------------

1. Откройте Google Sheets
2. File > Import > Upload
3. Загрузите файл habit_tracker_template.csv
4. После импорта нужно вручную:
   - Выделить ячейки с привычками (B6:AF100) и вставить чекбоксы:
     Insert > Checkbox
   - Создать график:
     Insert > Chart
     Выберите данные из блока статистики
     Тип графика: Area Chart
     Цвет: зеленый (#34a853)

ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ
----------------------

В скрипте есть дополнительные функции:

addHabit("Название привычки") - добавить новую привычку
removeHabit("Название привычки") - удалить привычку

Чтобы использовать их:
1. Откройте Apps Script
2. В консоли выполните: addHabit("Новая привычка")

НАСТРОЙКА
---------

Чтобы изменить месяц:
- В функции setupHabitTracker() измените значение переменной monthName

Чтобы изменить количество дней в месяце:
- В функции setupHabitTracker() измените значение переменной daysInMonth (30 или 31)
